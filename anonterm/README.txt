[?] Anonterm is a project set up to anonymize all network commands in the terminal, including launching firefox from your terminal, which is linked to the proxychains!

[?] The purpose of this project is to help keep privacy a right and to minimize personalized data collection.

[?] Command history is stored in a log.txt file.

[*] Please make sure to implement your own proxychains in your proxychains config file, usually located in the /etc directory for linux systems.

[*] Learn how to correctly add proxychains to that file and acivate the terminal!

[!] Test Anonterm to make sure it is configured properly!
